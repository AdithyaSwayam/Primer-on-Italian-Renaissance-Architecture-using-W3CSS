# Primer-on-Italian-Renaissance-Architecture-using-W3CSS
Italian Renaissance Architecture
A small multi-page website that presents landmark works of Italian Renaissance architecture through visually rich, content-focused HTML pages.

Overview
This project is a static website centered on five major case studies: Florence Cathedral, St. Peter's Basilica, Villa Rotonda, Palazzo Medici Riccardi, and Tempio Malatestiano.
 Each page combines historical context, key facts, architect profiles, architectural features, and significance notes in a consistent editorial layout.

Features
Multi-page structure with a shared navigation bar linking all architecture pages.

Responsive layout designed with custom CSS and mobile-friendly breakpoints.

Image sliders with thumbnail previews and next/previous controls on each monument page.

Reusable content sections such as Overview, Key Facts, Who Built It, Architectural Features, and Significance.

Typography using Google Fonts and light utility styling support from W3.CSS.

Tech stack
HTML5 for page structure.

CSS3 for layout, color, spacing, and responsive behavior.

Vanilla JavaScript for image slider interactions and thumbnail state updates.

External resources: Google Fonts and W3.CSS.
