# Primer-on-Italian-Renaissance-Architecture-using-W3CSS
# Italian Renaissance Architecture

A small multi-page website that presents landmark works of Italian Renaissance architecture through visually rich, content-focused HTML pages.[1][2][3][4][5]

## Overview

This project is a static website centered on five major case studies: Florence Cathedral, St. Peter's Basilica, Villa Rotonda, Palazzo Medici Riccardi, and Tempio Malatestiano.[1][2][3][4][5] Each page combines historical context, key facts, architect profiles, architectural features, and significance notes in a consistent editorial layout.[1][2][3][4][5]

## Features

- Multi-page structure with a shared navigation bar linking all architecture pages.[1][2][3][4][5]
- Responsive layout designed with custom CSS and mobile-friendly breakpoints.[1][2][3][4][5]
- Image sliders with thumbnail previews and next/previous controls on each monument page.[1][2][3][4][5]
- Reusable content sections such as Overview, Key Facts, Who Built It, Architectural Features, and Significance.[1][2][3][4][5]
- Typography using Google Fonts and light utility styling support from W3.CSS.[1][2][3][4][5]

## Pages

| File | Subject |
|---|---|
| `florence-cathedral.html` | Florence Cathedral, including Brunelleschi's dome and structural innovation.[5] |
| `st-peters.html` | St. Peter's Basilica, with Michelangelo's dome and Baroque additions.[1] |
| `villa-rotonda.html` | Villa Rotonda, focused on Palladian symmetry and villa design.[2] |
| `palazzo-medici.html` | Palazzo Medici Riccardi, a model of the Renaissance urban palace.[3] |
| `tempio-malatestiano.html` | Tempio Malatestiano, an early classical shell for a Christian building.[4] |
| `index.html` | Main entry page linking the individual building studies.[1][2][3][4][5] |

## Tech stack

- HTML5 for page structure.
- CSS3 for layout, color, spacing, and responsive behavior.[1][2][3][4][5]
- Vanilla JavaScript for image slider interactions and thumbnail state updates.[1][2][3][4][5]
- External resources: [Google Fonts](https://fonts.google.com/) and [W3.CSS](https://www.w3schools.com/w3css/).[1][2][3][4][5]

## Run locally

1. Clone or download the repository.
2. Keep all HTML files in the same root folder so the internal navigation links work correctly.
3. Open `index.html` in a browser.
